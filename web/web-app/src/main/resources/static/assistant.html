LEASE ASSISTANT

租房智能助手

这是 Spring AI 版助手的本地调试页，默认使用 SSE 流式输出，也可以切回普通 JSON 接口做对比。 当前助手采用“接口层 + 编排层 + 工具层 + 会话层”的拆分方式，页面会自动携带会话 ID，便于验证多轮上下文、工具调用和业务动作回执。

当前能力

  • 查询公寓详情、房间详情和房源列表
  • 查看我的浏览记录、预约记录和租约订单
  • 发起预约、取消预约、调整预约时间
  • 创建租约订单、查看订单详情、取消订单
  • 基于 Redis 保存会话上下文
  • 支持工具调用事件和流式回复展示
  • 当前版本未接入 RAG 知识库，重点验证业务工具链路
如果要查询“我的预约”“我的订单”等个人数据，需要先登录并填写 token。页面会自动保存到浏览器本地。
同一个会话 ID 会复用上下文历史。点击“新会话”会重新生成新的会话 ID，并清空当前聊天记录。
使用流式输出
开启后调用 /app/assistant/chat/stream，可观察增量回复、工具调用和工具返回；关闭后走普通 /app/assistant/chat 接口。
